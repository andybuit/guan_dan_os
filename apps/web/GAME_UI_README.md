# 掼蛋游戏界面

根据游戏截图实现的完整掼蛋游戏界面，包含所有交互功能。

## 已实现的功能

### ✅ 游戏组件

1. **GameCard** - 增强版卡牌组件
   - 支持选中/取消选中（上移动画）
   - 支持不同尺寸（xs, sm, md, lg）
   - 显示花色和点数
   - 显示配牌标识
   - 支持背面显示

2. **PlayerPosition** - 玩家位置组件
   - 显示玩家头像、昵称、金币
   - 显示剩余牌数
   - 显示最后出牌
   - 显示"不出"状态
   - 当前回合高亮
   - 倒计时显示

3. **HandCards** - 手牌区域组件
   - 卡牌选择功能（点击选中/取消）
   - 理牌按钮（按等级排序）
   - 提示按钮（智能选牌）
   - 显示当前选中数量
   - 卡牌重叠显示（节省空间）

4. **GamePage** - 主游戏界面
   - 四个玩家位置（上下左右）
   - 中央出牌区域
   - 手牌显示和选择
   - 出牌/不出/托管按钮
   - 回合管理和倒计时
   - 团队得分显示

### 🎮 交互功能

- ✅ 点击卡牌进行选择/取消选择
- ✅ 出牌按钮（移除手牌，显示在桌面）
- ✅ 不出按钮（跳过回合）
- ✅ 理牌按钮（按等级排序手牌）
- ✅ 托管按钮（自动出牌）
- ✅ 倒计时功能
- ✅ 回合轮换
- ✅ 实时UI更新

### 🎨 UI特性

- 使用青绿色主题（类似截图）
- 响应式布局
- 流畅的动画效果
- 清晰的状态指示
- 优雅的卡牌重叠效果

## 使用方式

访问游戏页面：
```
http://localhost:3000/game/room-123
```

或从房间页面点击"开始游戏"自动跳转。

## 文件结构

```
apps/web/
  app/
    game/[id]/
      page.tsx          # 主游戏页面
  src/
    components/
      game/
        GameCard.tsx     # 卡牌组件
        HandCards.tsx    # 手牌区域
        PlayerPosition.tsx # 玩家位置
```
